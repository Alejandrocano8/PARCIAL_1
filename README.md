# Proyecto Parcial

Este proyecto es una página web estática que simula la página de inicio de ExpressVPN. Está construida con HTML y CSS puro, sin JavaScript.

## Descripción

La página incluye:

- Un encabezado con el logo y selector de idioma
- Una sección hero con título y botón de llamada a acción
- Una sección de características destacando por qué elegir ExpressVPN
- Una sección de menciones en medios
- Un pie de página con enlaces y redes sociales

## Cómo usar

1. Abre el archivo `index.html` en tu navegador web.
2. La página se cargará y mostrará el contenido.

## Estructura del proyecto

- `index.html`: El archivo principal de la página web
- `stilos.css`: Los estilos CSS para la página
- `img/`: Carpeta con las imágenes utilizadas en la página

## Tecnologías utilizadas

- HTML5
- CSS3 (con Grid y Flexbox para el diseño responsivo)

No requiere instalación de dependencias ni servidor; es completamente estático.
